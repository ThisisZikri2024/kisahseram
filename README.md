# 🕸️ Arkib Mistik: Portal Kisah Seram

![Status](https://img.shields.io/badge/Status-Seram-red)
![Style](https://img.shields.io/badge/Theme-Glow%20LED-black)

**Arkib Mistik** adalah sebuah laman web ringkas (Single Page Application) yang direka khas untuk mengumpul, menyimpan, dan menguruskan koleksi kisah seram daripada orang ramai. 

## 🌙 Ciri-Ciri Utama
* **Glow LED Red Interface:** Rekaan bertemakan kegelapan dengan kesan cahaya neon merah.
* **Sistem Simpanan Tempatan:** Menggunakan `localStorage` supaya cerita tidak hilang walaupun browser ditutup.
* **Pengurusan Cerita:** Boleh menambah kisah baru dan memadam catatan yang tidak diperlukan.
* **Responsif:** Sesuai dibuka di komputer mahupun telefon pintar.

## 🛠️ Teknologi Digunakan
* **HTML5** - Struktur laman.
* **CSS3** - Kesan *Glow* dan animasi LED.
* **JavaScript (ES6)** - Logik simpanan data dan manipulasi DOM.

## 👻 Cara Penggunaan
1. Muat turun fail `index.html`.
2. Buka fail tersebut menggunakan mana-mana pelayar web (Chrome/Edge/Firefox).
3. Masukkan nama saksi dan huraian kejadian seram.
4. Tekan **Simpan ke Arkib**.
